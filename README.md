# Relatório Administrativo - Power BI 📊

## 📋 Sobre o Projeto
Este repositório contém um dashboard interativo desenvolvido no Power BI para análise de indicadores administrativos e financeiros. O objetivo é facilitar a tomada de decisão através de visualizações claras de KPIs.

## 🚀 Funcionalidades
* **Análise de Vendas:** Acompanhamento mensal e anual.
* **Gestão Financeira:** Visualização de custos e margem de lucro.
* **Banco de Dados:** Utiliza um arquivo SQLite (`vendas_db.sqlite`) para armazenamento dos dados.

## 🛠️ Tecnologias Utilizadas
* **Power BI** (Visualização de dados)
* **SQLite** (Banco de dados)
* **Git/GitHub** (Controle de versão)

## 📂 Como visualizar
1. Clone o repositório.
2. Certifique-se de ter o **Power BI Desktop** instalado.
3. Abra o arquivo do projeto e conecte ao banco de dados `vendas_db.sqlite` incluído na pasta.
