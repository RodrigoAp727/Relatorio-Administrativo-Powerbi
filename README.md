# Relatório Administrativo - Power BI 📊

## 📋 Sobre o Projeto
Este repositório contém um dashboard interativo desenvolvido no Power BI focado na análise de indicadores administrativos e financeiros. O objetivo é transformar dados brutos em insights visuais para facilitar a tomada de decisão.

## 🚀 Funcionalidades e KPIs
* **Visão Geral:** Acompanhamento de métricas principais (Vendas, Custos, Lucro).
* **Análise Temporal:** Comparativo de desempenho mensal e anual.
* **Banco de Dados:** Conexão direta com banco de dados SQLite para maior agilidade.

## 🛠️ Tecnologias Utilizadas
* **Power BI** (Criação de Dashboards e tratamento de dados com Power Query)
* **SQLite** (Armazenamento e estruturação de dados)
* **GitHub** (Controle de versão e documentação)

## 📂 Como visualizar este projeto
1. Baixe o arquivo do projeto (.pbix) e o banco de dados deste repositório.
2. Certifique-se de que o **Power BI Desktop** está instalado em sua máquina.
3. Ao abrir o projeto, caso peça a fonte de dados, aponte para o arquivo `vendas_db.sqlite` baixado.
